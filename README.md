# vhu
virtual host ubuntu

create automatic apache2 virtual host in ubuntu by vhu.

example to run code:
http://example.com/vhu.php?site=test.dev&mail=me@sarbazk.com&dr=/var/www/html/test&pass=***

get method variables means:
site = SiteName
mail = SiteAdmin
dr = DocumentRoot
pass = your system(ubuntu) password
